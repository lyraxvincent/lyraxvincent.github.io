  • Python Package datastand

Datastand is a python package designed to help Data Scientists, Machine Learning Engineers and Data Analysts to better understand data. It gives quick insights about given data; general dataset statistics, size and shape of dataset, number of unique data types, number of numerical and non- numerical columns, small overview of dataset, missing data statistics, missing data heatmap and provides methodology to impute missing data.
